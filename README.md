# 기능 목록
**1. [입력] 문자열을 입력받는다.**

**2. [중간 과정] 커스텀 구분자를 추출한다.**
   - [예외 처리] //와 \n 중 하나만 존재할 경우 예외를 발생시킨다.

**3. [중간 과정] 커스텀 구분자를 제외한 순수 계산식을 추출한다.**

**4. [중간 과정] 문자열을 등록된 구분자로 분리한다.**
   - [예외 처리] 숫자가 없고 문자만 존재하는 경우 예외를 발생시킨다.
   - [예외 처리] 등록된 구분자 외의 문자가 존재하는 경우 예외를 발생시킨다.
   - [예외 처리] 음수가 존재하는 경우 예외를 발생시킨다.

**5. [중간 과정] 분리된 문자들을 다 더한다.**
   - [예외 처리] 숫자가 아닌 문자가 존재하는 경우 예외를 발생시킨다.

**6. [출력] 덧셈 결과를 출력한다.**

---
# 정상 케이스
**1. 커스텀 구분자가 없는 문자열**
   - 1,2:3
   - 123

**2. 커스텀 구분자가 존재하는 문자열**
   - //;\n1;2:3,4
   - //;\n123

# 예외 케이스
**1. 커스텀 구분자가 없는데, 문자열 사이에 미등록 구분자가 존재하는 경우**
  - 1,2;3

**2. 커스텀 구분자가 존재하고, 문자열 사이에 미등록 구분자가 존재하는 경우**
  - //;\n1%2%3

**3. 커스텀 구분자가 //와 \n 사이에 존재하지 않는 경우**
  - ///&\n1,2,3
  - //&\m1,2,3
  - /*\nm1,2,3

**4. 숫자가 없고 문자만 존재하는 경우**
  - ;,,
  - //*\n***
